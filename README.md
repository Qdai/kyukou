## For development

1. Set environment variables: `NODE_ENV`, `PORT` and `TZ`
2. Configure: `cp .env.example .env` then edit `.env`
3. Install: `NODE_ENV=development npm install`
4. Build: `npm run build`
5. Start: `npm start`
